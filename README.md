## Getting Started
To get started with VoxaLive-AI, follow these simple steps:
1. Clone the repository:  
   ```bash
   git clone https://github.com/kavirajannv/VoxaLive-AI.git
   ```  
2. Install dependencies:  
   ```bash
   cd VoxaLive-AI
   npm install
   ```
3. Run the application:  
   ```bash
   npm start
   ```
