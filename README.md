## 🧠 Project Overview

**VoxaLive-AI** is a real-time AI-powered communication platform that enables seamless interaction between users speaking different languages.

Users can **chat, audio call, video call, and join meetings**, where everything is automatically translated into their **preferred native language**.

For example:

* A user speaks **Tamil** → the other user hears **English**
* The other user replies in **English** → it is translated back to **Tamil**

This works in:

* 💬 Chat
* 📞 Audio Calls
* 🎥 Video Calls
* 👥 Group Meetings / Classrooms

---

## 🎯 Problem Statement

In real-world communication, especially in classrooms, meetings, and global collaboration, people face language barriers.

Example:

* Teachers speak English
* Students understand only Tamil
* Communication becomes difficult

---

## 💡 Solution

VoxaLive-AI solves this by:

* Translating conversations **in real-time**
* Supporting **multi-user meetings**
* Generating **automatic AI meeting notes**
* Allowing users to communicate in their **own native language**

## 🤖 How It Works

1. User joins chat / call / meeting
2. Voice or text is captured
3. Speech is converted to text (Speech-to-Text)
4. NLP processes the content
5. AI translates into target language
6. Output is delivered instantly
7. Meeting notes are generated automatically

## 🔐 Environment Variables

Create a `.env` file inside backend folder:

PORT=5000
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret
API_KEY=your_translation_api_key

⚠️ Do NOT upload `.env` file to GitHub
